To complete both task, I created a fully functional and interactive dashboard using Power BI.
First, I collected and prepared the dataset. I ensured that the data was clean, with no missing or incorrect values. Then, I opened Power BI Desktop and used the “Get Data” option to import the dataset into the tool.
After loading the data, I moved to the Data View to check and correct data types. For example, I made sure numerical fields like sales, profit, and orders were set correctly so calculations would work properly.
Next, I started building the dashboard. I created multiple visuals to represent the data clearly:
KPI Cards to show key values like Total Sales, Total Profit, Total Orders, and Total Customers
A Line Chart to display sales trends over time
A Bar Chart to compare sales across different regions
A Donut/Pie Chart to show category-wise sales distribution
Additional charts for customer segments and top-performing products
I used the drag-and-drop feature in Power BI to place fields into visuals. Then, I added filters and slicers (such as region and date) to make the dashboard interactive, allowing users to explore the data easily.
After creating the visuals, I arranged them neatly on the dashboard and applied formatting like colors, titles, and labels to make it user-friendly and visually appealing.
Finally, I analyzed and add the key insights, such as:
Sales growth compared to previous periods
Best-performing categories and regions
Customer segments contributing the most revenue
